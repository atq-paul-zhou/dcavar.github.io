Other activities

Community services

LINGUIST List

  • Co-director: among others moderating, system administration (AWS servers), software development and maintanance, personnel management, relations with supporting publishers, advertisers, partners.
  • I conceptualized and realized also these services:
    • GeoLing (mapping linguistic institutions, events, and other relevant information on a global map). This is a real-time mapping of LINGUIST List posts on a global map.
    • Alexa (Amazon Echo) Flash Briefing on LINGUIST List. Go to your Alexa App and set up a new Flash Briefing. You can hear updates on Jobs, Book announcements, Conference calls, etc. using Alexa.
    • LINGUIST List postings on Facebook, Google+, Twitter.
    • Full migration of LINGUIST List to AWS using Linux VMs, open source PostgreSQL and web services, Mailman list server, online donation and payment systems, etc.
